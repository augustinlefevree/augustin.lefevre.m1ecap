# augustin.lefevre.m1ecap

Package R contenant un jeu de données de football et des fonctions d'analyse.

## Description

Ce package permet d'analyser des matchs de football à partir d'un jeu de données contenant des résultats, statistiques et cotes.

## Fonctions principales

- filtrer_resultat() : filtre les matchs selon le résultat
- calcul_buts_moyens() : calcule la moyenne des buts par équipe
- plot_buts_moyens() : affiche un graphique des buts moyens
